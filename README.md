# Le Jardinier Sarthois

Site web de l'association "Le Jardinier Sarthois"

## Structure du projet

```
le_jardinier_sarthois/
├── .dev/                       # Documentation et gestion de projet
│   ├── requirements.md         # Cahier des charges
│   └── todo.md                 # Suivi des tâches
├── src/                        # Code source du site (futur)
├── public/                     # Fichiers publics du site
│   ├── images/                 # Images optimisées pour le web
│   │   └── originals/          # Backup des images originales
│   └── downloads/              # Fichiers téléchargeables
├── Documents/                  # Documents métier et administratifs
└── tools/                      # Scripts utilitaires de développement
    ├── README.md               # Documentation des outils
    └── optimize_images.py      # Script d'optimisation d'images
```

## Documentation

- **[Cahier des charges](.dev/requirements.md)** : Spécifications et historique du projet
- **[Suivi des tâches](.dev/todo.md)** : Planning et avancement du développement
- **[Outils de développement](tools/README.md)** : Documentation des scripts

## Installation

### Prérequis
- Python 3.12+
- Pillow : `pip install Pillow`

### Utilisation des outils
Voir [tools/README.md](tools/README.md) pour la documentation complète des scripts.

## Bonnes pratiques

- Toujours optimiser les images avant de les ajouter au site
- Les backups sont automatiques mais vérifier leur présence
- Consulter le cahier des charges avant toute modification
- Mettre à jour la documentation au fur et à mesure
